# FaceForge

Modular, production-ready face detection and recognition toolkit.

Author: Developer6316

Quick start

1. Clone the repo and install requirements:

   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt

2. Download models (see colab/models_download.sh) and run the demo:

   python scripts/demo_webcam.py

Repository layout

- src/faceforge/: core package
- scripts/: command-line utilities
- colab/: Colab notebooks and model download scripts
- tests/: pytest tests
- docs/: design and usage docs
- docker/: GPU-ready Dockerfile
